## 其它的设置   
```
     加载图片
     {
        test: /\.(gif|jpg|jpeg|png|svg)$/,   图片的格式
        use: [{
            loader: 'url-loader',            
            options: { 
            limit: 1024,                    大小转换
            name: '[name]-aaa.[ext]'       文件的名字[ext:扩展名字]
            }
        }]
     }
     加载CSS
     {
         test:/\.css$/
         use:[
             'style-loader', //插入到html中
             'css-loader'    //加载到
         ]
     }
    创建一个style文件夹，产能构建一个css文件，然后在使用的地方就可以引入它
