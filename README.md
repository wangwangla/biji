## 介绍

仓库主要记录平时学习中遇到的问题，以及每天学习的一下笔记，包含基础和一些常见问题的解决方法，更多是最近关注有关软件的东西，也包括平时看过的书本笔记。希望它是自己每天生活的一点总结。



之前主要是学生，一般都是学习相关的东西，如今入职，以后就每天抽出时间，来讲每一个模块重新进行编写和完善。
